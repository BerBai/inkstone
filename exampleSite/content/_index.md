Welcome to **Inkstone** — a minimal bilingual Hugo theme designed for longreads, slow writing, and CJK-friendly typography.

This is a demo site running entirely on the theme's `exampleSite/` directory. Browse the [posts](/posts/), check the [archive](/archive/), or explore the full [shortcodes reference](/shortcodes-demo/).

## What's inside

- **Bilingual i18n** — English / 中文 out of the box, fully translated UI strings
- **CJK typography** — Auto-detection, character-based reading time, optimized line-height for mixed scripts
- **25+ content shortcodes** — admonitions, callouts, galleries, video embeds, tabs, mermaid/markmap diagrams, more
- **Tailwind v4** — Using Hugo 0.128+ built-in `css.TailwindCSS`, no separate build step needed
- **Pagefind search** — Optional, opt-in via your build pipeline (`<C-k>` modal already in theme)
- **Dark mode** — Respects `prefers-color-scheme`, toggle persists to `localStorage`
